# Claude Code Plugin Marketplace

Collection of Claude Code plugins(LSP/MCP/commands/hooks/skills/subagents).

## Installation

1. Install marketplace: `claude plugin marketplace add gdm257/cc-plugins`
2. Install plugin on-demand: `claude plugin`

## Available Plugins

| Plugin                  | Dependencies                                              | Type         |
| ----------------------- | --------------------------------------------------------- | ------------ |
| **php-lsp-bunx**        | `bunx`                                                    | LSP (PHP)    |
| **php-lsp-npx**         | `npx`                                                     | LSP (PHP)    |
| **php-lsp**             | `intelephense`                                            | LSP (PHP)    |
| **pyright-lsp-uvx**     | `uvx`                                                     | LSP (Python) |
| **pyright-lsp-pipx**    | `pipx`                                                    | LSP (Python) |
| **ty-lsp-uvx**          | `uvx`                                                     | LSP (Python) |
| **ty-lsp-pipx**         | `pipx`                                                    | LSP (Python) |

| **typescript-lsp-npx**  | `npx`                                                     | LSP (JS/TS)  |
