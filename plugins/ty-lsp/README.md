# ty-lsp

Python language server (ty) for type checking (mypy) and code intelligence (pyright).

## Supported Extensions
`.py`, `.pyi`

## Installation

Install ty globally via uv:

```bash
uv tool install ty
```

Or with pip:

```bash
pip install ty
```

Or with pipx (recommended for CLI tools):

```bash
pipx install ty
```
